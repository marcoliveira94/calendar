# Calendar

Calendar is a React app.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) or [npm](https://www.npmjs.com/)  to install.

```bash
yarn || npm install
```

## Run


```bash
yarn start || npm run start
```

## Run Tests
```bash
yarn test || npm run test
```
